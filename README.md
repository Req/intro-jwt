## JWT

- JWT is a standard
- JSON WEB TOKEN
- Token will look like AAAAAAA.BBBBBB.CCCCCC
- [Header].[Payload].[Signature]
- Secure transmission of data by signing data with a secret
- Secret must be kept secret!
- Token must be stored securely!
